# FocusFlow

Cronômetro Pomodoro desenvolvido com HTML, CSS e JavaScript.

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Timer Pomodoro
- Pausa Curta
- Pausa Longa
- Barra de progresso circular
- Estatísticas de sessões
- Som de alerta
- LocalStorage
- Responsividade

## Conceitos Trabalhados

- setInterval
- clearInterval
- Manipulação do DOM
- Eventos
- SVG
- LocalStorage
- Temporizadores

## Modos

- Pomodoro (25 minutos)
- Pausa Curta (5 minutos)
- Pausa Longa (15 minutos)
